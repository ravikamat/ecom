Getting Started

Opening the App

ECO Command Center v2 requires a local Node.js backend to power web scraping, database persistence, and AI features.

First-Time Setup

  1. Install Node.js: Ensure you have Node.js (v18+) installed on your machine.
  2. Install Dependencies: Open a terminal in the project directory and run npm install.
  3. Start Server: Run node server.js to start the backend on port 3000.
  4. Access the App: Open http://localhost:3000 in your browser.
  5. Configure API Keys: Click the Settings (⚙️) icon in the top right. Enter your NVIDIA NIM API keys for the primary (GLM-5.2) and fallback (MiniMax-M3) AI models. Keys are securely stored in your local SQLite database.
  6. Select Region: Choose your default country and currency in the top bar to localize pricing and supplier searches.

Deep Dive: Codebase Architecture & Files

ECO Command Center v2 is structured as a monolithic Node.js backend serving a lightweight, dependency-free vanilla JavaScript frontend. This ensures ultra-fast performance, localized privacy, and the ability to bypass cross-origin browser restrictions during web scraping.

🖥️ Backend Architecture

  • server.js (2,300+ lines): The heart of the application.
    • Central CONFIG: Manages all caching TTLs, rate limits (via sliding window), and request timeouts.
    • API Routing: Exposes RESTful endpoints (/api/trending/page, /api/ai-status, /api/db/settings).
    • Deep Research Controller: Orchestrates the multi-round AI and web scraper feedback loop.
    • AI Shim: Manages HTTP requests to the NVIDIA NIM endpoint, parsing SSE streams, executing server-side tool calls, and handling the auto-fallback logic between GLM-5.2 and MiniMax-M3.
  • db/sqlite.js:
    • Uses better-sqlite3 (or equivalent driver) to initialize the eco.db file locally.
    • Defines strict schemas for Settings (key-value pairs) and SavedItems.
    • Exposes direct CRUD operations to server.js for secure local persistence.

🌐 Frontend Architecture (Client-Side)

All frontend scripts are located exclusively in the /js/ directory to ensure strict separation of concerns:

  • app.js: Bootstraps the application, restores user sessions/themes, manages SPA routing, and handles the Settings Modal logic (secure dual API key loading and testing).
  • trending.js: Maps JSON arrays of products into HTML. Maintains _trendPage state, utilizes an IntersectionObserver for infinite scrolling and background pre-fetching, and uses fuzzy string matching to visually flag already-saved products.
  • search.js: Handles the complex multi-platform query string builder and parses cross-origin JSON responses (proxied through the backend) into price comparison tables.
  • dashboard.js: Ingests aggregated backend statistics and utilizes null coalescing to render resilient charting data.
  • calculator.js: Contains the proprietary FinancialEngine logic. Listens for DOM input events to recalculate True Cost, Return Reserves, and ROI in real-time.
  • saved.js: The presentation layer for the SQLite database. Implements sorting, filtering, and deleting of bookmarked items.
  • chatbot.js: Connects to the backend via Server-Sent Events (SSE). Parses JSON streams into typing indicators, raw text, and renders rich HTML widgets (Product Cards and Price Tables) via the _renderToolResult() function.
  • db.js: A legacy wrapper refactored to act as an asynchronous HTTP client proxy to the new SQLite REST endpoints, ensuring backward compatibility.
  • ai-engine.js: Client-side helpers for formatting chat histories and separating system vs. user prompt roles.
  • competitor-tracker.js & suppliers.js: Specialized modules for pricing elasticity and rendering B2B supplier contact cards.

🎨 Styling & DOM Layout

  • index.html: The master structural template. Contains all <section> wrappers, the Top Bar, the Settings Modal, and the Floating Chat widget.
  • css/styles.css: A premium, unified dark-theme design system using CSS Variables. Contains specific logic for glassmorphism, shimmer loading states (#btn-deep-research), status indicator dots, and responsive grid layouts.

Dashboard

The Dashboard is your command center's home screen. It gives you a bird's-eye view of your research activity and quick access to key features.

Stat Cards

At the top of the Dashboard, four stat cards display real-time summaries of your activity:

📦 Saved Products

The total number of products, suppliers, and calculations you've bookmarked in the Saved List. This updates automatically as you add or remove items.

🏭 Suppliers Found

The total number of unique suppliers you've discovered and interacted with across all searches.

📊 Avg. Margin

The average profit margin across your saved calculations. A quick health-check on the viability of products you're evaluating.

💰 Working Capital

An estimate of the total upfront investment needed for all your saved products, based on MOQ × supplier base price + shipping costs.

Market Snapshot Table

The market snapshot shows the top trending products for your selected country at a glance. Each row displays the product name, category, demand score, competition level, and average selling price. Click any product to jump directly to the Search & Compare tab with that product pre-filled.

Quick Action Cards

Below the market snapshot, quick action cards let you jump to the most-used workflows:

  • Find Products — Opens the Trending Products tab
  • Compare Prices — Opens the Search & Compare tab
  • Calculate Costs — Opens the Cost Calculator tab
  • Source Suppliers — Opens the Supplier Finder tab

AI Market Insight

Click the "Get AI Insight" button on the Dashboard to receive an AI-generated summary of the current market landscape for your selected country. The AI analyzes trending categories, demand patterns, and provides actionable recommendations tailored to your market.

💡 Tip

The Dashboard stats update in real-time. Keep it open as a "home base" to track your research progress.

Trending Products

The Trending Products tab is your product research engine. It surfaces high-demand products across categories and geographies, helping you spot opportunities before they become saturated.

Filtering Products

Use the filter controls at the top to narrow your results:

  • Country — Filter by market. Defaults to your selected country, but you can explore other markets too.
  • Category — Filter by product category (Electronics, Fashion, Home & Kitchen, Beauty, Sports, Toys, etc.).
  • Sort Order — Sort by demand score (high to low), competition level, or price range.

Understanding Demand Scores

Each product has a Demand Score rated from 0 to 100:

Score RangeLabelMeaning
80–100🔥 Very HighExtremely popular; high search volume and sales velocity
60–79📈 HighStrong demand with room for new sellers
40–59📊 MediumModerate demand; evaluate competition carefully
20–39📉 LowNiche product; may need targeted marketing
0–19⬇ Very LowLimited demand; high risk for new sellers

Understanding Competition Levels

Competition is rated as Low, Medium, or High based on the number of existing sellers, brand dominance, and price pressure in the category. The ideal product has high demand and low competition.

Saving Products

Click the bookmark icon (🔖) on any product card to save it to your Saved List. You'll see a confirmation toast notification. Saved products can be reviewed, annotated, and exported from the Saved List tab.

AI Discover Feature

Click "AI Discover" to have the AI suggest untapped product opportunities based on your selected country and filters. The AI looks for gaps in the market — products with rising demand but limited competition.

💡 Tip

Look for products with a demand score above 60 and competition rated "Low" or "Medium" — these are your golden opportunities.

Search & Compare

Search for any product and instantly see a comparison of fees, projected profit, and margins across all major e-commerce platforms in your country.

How to Search

Type a product name or keyword into the search bar and press Enter or click the search button. Results are generated based on your selected country's platform ecosystem. You can also use the global search shortcut Ctrl + K from anywhere in the app.

Platform Comparison Table

The results table shows a side-by-side comparison across platforms (e.g., Amazon, Flipkart, Meesho, eBay, Walmart, etc. — depending on your country). For each platform, you'll see:

  • Commission % — The percentage the platform takes as a referral or commission fee.
  • Fixed Fees — Any flat-rate fees per order (closing fee, shipping subsidy deduction, etc.).
  • Estimated Profit — Your projected profit after all platform fees.
  • Profit Margin % — The net margin as a percentage of your selling price.

Matched Suppliers

Below the comparison table, the app shows matched suppliers — vendors that carry the product you searched for (or close alternatives). Each supplier card displays their name, type, location, MOQ, and rating.

AI Smart Search

Click "AI Smart Search" for an AI-enhanced analysis of the product. The AI provides market insights, pricing strategy recommendations, and highlights the best platform for selling the product based on fee structures and category performance.

Saving Search Results

Click "Save Result" to bookmark the entire search result — including the comparison data and matched suppliers — to your Saved List for later review.

💡 Tip

Compare margins across 3–4 platforms before choosing where to list. Even a 2–3% difference in commission can significantly impact profitability at scale.

Supplier Finder

The Supplier Finder helps you discover and evaluate sourcing partners across multiple categories and regions.

Searching & Filtering

Enter a product keyword to find relevant suppliers. Use the filter controls to narrow results by:

  • Supplier Type — Manufacturer, Wholesaler, Dealer, or Dropshipper
  • Rating — Minimum star rating (1–5)
  • Location — Domestic or international suppliers

Understanding Supplier Types

🏭 Manufacturer

Produces goods directly. Usually offers the lowest per-unit cost but requires higher MOQs. Best for established sellers with capital to invest.

📦 Wholesaler

Buys in bulk from manufacturers and sells in smaller quantities. Moderate MOQs with reasonable pricing. A good balance for growing sellers.

🏪 Dealer / Distributor

Regional or authorized distributors. May offer brand-name products with lower MOQs. Prices are higher per unit but often include support and logistics.

🚀 Dropshipper

Ships directly to your customers — no inventory needed. Zero MOQ but lowest margins. Ideal for testing new products with minimal risk.

MOQ Explained

Minimum Order Quantity (MOQ) is the smallest number of units a supplier will sell per order. Lower MOQs mean less upfront capital but may result in higher per-unit costs. Always factor MOQ into your working capital calculations using the Cost Calculator.

Rating System

Suppliers are rated on a 1–5 star scale based on reliability, response time, product quality, and consistency. Ratings above 4.0 are considered excellent. Sort by rating to find the most trusted suppliers first.

AI Suggest Suppliers

Click "AI Suggest" to let the AI recommend the best suppliers for your product based on your budget, target margin, and country. The AI considers factors like supplier reliability, MOQ, pricing, and logistics feasibility.

💡 Tip

Always contact 2–3 suppliers before committing. Compare MOQs, sample availability, and shipping terms to find the best fit for your business model.

Cost Calculator

The Cost Calculator is the most detailed tool in the Command Center. It calculates exact profit, margin, break-even, and working capital requirements for any product — accounting for every cost from sourcing to delivery.

Input Fields Walkthrough

  1. Supplier Base Price — The per-unit cost from your supplier, in your selected currency. This is the foundation of your entire calculation.
  2. Minimum Order Quantity (MOQ) — How many units you'll order from the supplier. This affects your total investment and working capital calculation.
  3. Inward Shipping — The cost to get inventory from the supplier to your location (or warehouse), expressed per unit. Include freight, customs, and handling charges.
  4. GST / VAT / Tax Rate (%) — The applicable tax rate on the product in your country. The calculator uses this to compute tax payable and input tax credit.
  5. Input Tax Credit (ITC) — Toggle this on if you can claim tax credits on your purchases (common for GST-registered businesses in India). When enabled, the tax paid on supplier purchases is offset against tax collected on sales.
  6. Platform Selection — Choose the e-commerce platform you plan to sell on. The calculator automatically applies the correct commission rates, closing fees, and shipping subsidies for that platform.
  7. Selling Price — Your target selling price per unit (including tax). Experiment with different prices to find the sweet spot between competitiveness and profitability.
  8. Packaging Cost — Per-unit cost for packaging materials (boxes, poly bags, bubble wrap, labels, etc.).
  9. Return Rate (%) — The estimated percentage of orders that will be returned. Industry averages range from 2–5% for most categories, but fashion can be 15–20%.
  10. Advertising / Marketing — Per-unit allocation for paid ads (PPC, sponsored listings, social media ads). Even a small ad budget can dramatically impact margins.
  11. Miscellaneous Costs — Any other per-unit costs not covered above (photography, warehousing, software subscriptions, etc.).

Reading the Results

After entering all fields, the calculator instantly computes:

MetricWhat It Means
Net Profit per UnitYour actual take-home profit after every cost and fee is deducted
Profit Margin %Net profit as a percentage of selling price — aim for 15–30% minimum
Break-Even UnitsThe number of units you need to sell to recover your initial investment
Working CapitalTotal upfront investment needed: (Base Price + Inward Shipping) × MOQ + initial costs
ROI %Return on investment — how much profit per rupee/dollar invested

Cost Breakdown

The visual cost breakdown chart shows exactly where your selling price goes — supplier cost, platform fees, taxes, packaging, shipping, ads, and profit. This helps you identify the biggest cost drivers and optimize them.

AI Optimize Feature

Click "AI Optimize" to receive AI-powered suggestions for improving your margins. The AI may suggest alternative pricing strategies, packaging optimizations, or platform switches to increase profitability.

Saving Calculator Results

Click "Save Calculation" to bookmark the current result. Saved calculations retain all inputs and outputs, so you can revisit and compare different product scenarios from the Saved List tab.

⚠️ Important

The calculator provides estimates based on standard platform fee structures. Actual fees may vary based on your seller plan, category-specific rates, and promotional offers. Always verify with the platform's official fee calculator before finalizing.

Saved List

The Saved List is your personal research vault. Everything you bookmark — products, suppliers, search comparisons, and cost calculations — lives here for easy reference.

Managing Saved Items

Each saved item shows a summary card with the key details. Items are categorized by type (Product, Supplier, Comparison, Calculation) and sorted by the date saved, with the most recent items first.

Adding & Editing Notes

Click the pencil icon (✏️) on any saved item to add or edit a personal note. Notes are great for recording your analysis, next steps, or reasons why you saved the item.

Deleting Items

Click the trash icon (🗑️) to remove an item from your saved list. You'll see a confirmation prompt before deletion to prevent accidental removals.

CSV Export

Click "Export CSV" to download your entire saved list as a CSV file. The export includes all item details, your notes, and timestamps — perfect for analysis in spreadsheet software like Excel or Google Sheets.

Bulk Actions

Use the checkboxes to select multiple items, then use the bulk action toolbar to:

  • Delete Selected — Remove multiple items at once
  • Export Selected — Export only the checked items to CSV
💡 Tip

Use notes to tag items with labels like "HIGH PRIORITY" or "FOLLOW UP" so you can quickly scan your saved list for actionable items.

Country & Currency

The Command Center supports multiple countries and currencies, letting you research products and costs in any market.

Global Country Selector

The country selector in the top header is global — changing it updates the entire app. Trending products will show data for the new country, platform fees adjust to local marketplaces, and supplier search prioritizes relevant regions.

Currency Conversion

All prices, costs, and calculations throughout the app are displayed in your selected currency. The app uses live exchange rates fetched from a public API, so figures are always up-to-date.

Independent Currency Selection

You can set the currency independently of the country. For example, you could research products in India but display all prices in USD for easier comparison. Use the currency dropdown in the header next to the country selector.

Exchange Rate Caching & Updates

Exchange rates are cached locally for offline use. Rates are refreshed automatically when the app detects an internet connection. The cache duration is typically 24 hours to balance freshness with API rate limits.

💡 Tip

When comparing products across countries, keep the currency consistent (e.g., always use USD) so you can make accurate cross-border comparisons.

  • Try different markets — Switch countries and run the AI again to get market-specific insights for cross-border opportunities.
  • Use as a starting point — AI suggestions are generated estimates. Always validate with your own research and real supplier conversations.

Understanding AI Limitations

⚠️ Important

AI-generated insights are estimates and suggestions, not guarantees. The AI model provides directional guidance based on patterns and general market knowledge. Actual market conditions, supplier pricing, and platform fees may differ. Always verify critical business decisions with primary data sources.

Additional limitations to be aware of:

  • AI responses require an active internet connection
  • Response times vary (typically 3–10 seconds depending on complexity)
  • The AI may occasionally produce outdated pricing or fee information
  • Rate limits may apply during peak usage — wait a moment and try again

FAQ & Troubleshooting

Common questions and solutions for issues you might encounter.

All your data — saved products, suppliers, calculations, notes, and preferences — is stored locally in your browser using IndexedDB and LocalStorage. Nothing is sent to any external server (except AI queries, which require an internet connection). Your data stays private on your device.
To reset all data, open your browser's Developer Tools (F12), go to the Application tab, and clear both LocalStorage and IndexedDB entries for the page. Alternatively, clear the site data from your browser's settings. This action is irreversible — consider exporting your saved list to CSV first.
The app caches exchange rates for offline use. If rates seem stale, check your internet connection — the app will automatically fetch fresh rates when connectivity is restored. Cached rates are typically refreshed every 24 hours. If the issue persists, clear the site's LocalStorage and reload the page.
AI features require an active internet connection. If the AI isn't responding, check the following:
  • Internet connection — Ensure you're online.
  • CORS issues — If running from a local file (file://), some browsers block API requests. Try serving the file via a local HTTP server (e.g., npx serve or Python's http.server).
  • Rate limits — The NVIDIA API has usage limits. If you've made many requests in a short time, wait a few minutes and try again.
  • API key — Ensure the API key embedded in the app is valid and hasn't been revoked.
The app is optimized for modern Chromium-based browsers:
  • Google Chrome (recommended) — Best performance and full feature support
  • Microsoft Edge — Fully supported
  • Mozilla Firefox — Fully supported
  • Safari — Mostly supported; some minor CSS differences may occur
  • Internet Explorer — Not supported
Yes! The app is fully responsive and works on mobile browsers. On smaller screens, the tab navigation switches to a scrollable horizontal layout, and some tables become horizontally scrollable. For the best experience, we recommend a tablet or desktop for detailed cost calculations.
